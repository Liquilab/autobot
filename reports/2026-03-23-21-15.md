## Wat heb ik geleerd
1. Polymarket gebruikt **USDC.e** (bridged USDC), niet native USDC - we moeten onze USDC swappen
2. De CLOB is off-chain maar settlement is on-chain op Polygon
3. API authenticatie werkt met EIP-712 signing → HMAC API keys
4. Sportmarkten hebben het hoogste volume en dagelijkse resolutie
5. We hebben POL nodig voor gas transacties
